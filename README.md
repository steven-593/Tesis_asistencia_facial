# Sistema de Asistencia Facial (Tesis)

Sistema web para el control de asistencia universitario mediante reconocimiento facial biométrico.

## Tecnologías
* **Frontend:** React + Vite
* **Backend:** PHP (Nativo)
* **Base de Datos:** MySQL
* **IA:** Face-API.js

## Instalación
1. Clonar el repositorio.
2. Importar la base de datos `asistencia_db.sql`.
3. Configurar la conexión en `backend/config/base_datos.php`.
